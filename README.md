# E-Commerce API

## Packages used

- Express
- JOI
- mongoose

### Getting Started - Install all dependencies

```
Run [`npm install`] in your command line
```

### Auth Route

- Login

- Update Password (Authenticated)
- Get All Users (Authorized & Authenticated)
- Get Single User
- Update User
